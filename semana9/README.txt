Semana 9.
En esta semana se subieron archivos sobre el uso de lugares en la memoria y funciones creadas por el programador.
En los archivos cuyos nombres terminan en "función", se hace un programa donde se calcula el cuadrado de un número dado por el usuario, usando distintos tipos de funciones, con argumento de salida, o de entrada, con o sin ninguno de ellos.
En el archivo "Funciones.c" se hizo un programa que usa 4 tipos distintos de funciones para pedir números y sacar algunos valores a raiz de ellos.
En el archivo "Mayor.c" se hizo un programa que toma los números de un archivo externo y, usando apuntadores, imprime el de mayor valor.
En el archivo "Promedio.c" se hizo un programa similar al anterior, pero este usa los números del archivo externo para realizar un promedio de ellos.
